# agrinhoJoseMarcelo
# Agro Forte, Futuro Sustentável - Agrinho 2026

> Projeto focado no equilíbrio entre a produção agrícola de alta eficiência e a manutenção/preservação do meio ambiente.

## 🛠️ Tecnologias Utilizadas
* **HTML5:** Estrutura semântica rigorosa.
* **CSS3:** Estilização com Flexbox layout, transições suaves, variáveis nativas e total responsividade (Mobile, Tablet e Desktop).
* **JavaScript Puro (Vanilla):** Manipulação dinâmica do DOM, troca de temas (Dark Mode), simulação interativa de impactos e validação lógica de formulário.
